< UserDataService : 유저데이터 서비스 >

*유저데이터 서비스는 현제 로그인한 유저의 데이터 정보를 가지고 있는 객체로 다음 정보를 가지고 있습니다.

name : 유저 이름
email : 유저 이메일
password : 유저 비번
resevation : 예약 정보 { cafeId , seatNumber, time }

유저데이터 서비스는 다음과 같이 이용합니다.
생성시

var 변수 = new UserDataService(); // 객체를 생성합니다.
변수.loadUserId();  // 유저 ID를 업데이트 합니다.
변수.getUserProfile();  //유저 프로필을 업데이트합니다.
